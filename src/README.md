# src
静的フロントエンド各アプリ。フォルダ名は2文字の符牒(コードネーム)で、対応表は下記の通り。
`common/`のみ例外で、個別アプリではなく全アプリ共通の基盤のため実名にしている。

| コード | 内容 |
|---|---|
| `ba` | 路歴(ろれき) — 追記オンリー・赤黒帳票のログ台帳。n4後継。Googleログイン必須、Claude Code用にPC/スマホ別鍵のレーンを持つ |
| `bb` | `ba`の「現在形ビュー」— `ba`の追記ログから今の状態を計算して見せる読み取り専用の投影ページ(ログイン不要) |
| `bc` | CASL2/COMET2 アセンブラ・エミュレータ |
| `bd` | ふっかつのじゅもん解析器(ドラクエ1、じゅもん→ステータス) |
| `be` | `n1`のスコア記録を折れ線グラフで見せる読み取り専用ページ(Googleログイン必須) |
| `bf` | ふっかつのじゅもん生成器(ドラクエ2、ステータス→じゅもん) |
| `bg` | `ba`用のYML定義エディタ |
| `bh` | NASM x86 エミュレータ(ミニアセンブラ+簡易x86仮想機械、同梱ROMなし) |
| `common` | 全アプリ共通の設定/認証/CSS (`config.js`, `auth.js`, `base.css`, `ledger.css`) |
| `k1` | NESエミュレータ(同梱ROM入り) |
| `k2` | `ba`のスレッドを投稿者別/分類別に集計するレーダーチャート |
| `n1` | 日々のスコア記録・一覧(Googleログイン必須) |
| `n2` | 大阪市・東大阪の地図(GeoJSON) |

各アプリのAPI呼び出し先は[`api/`](../api/README.md)（`common/config.js`の`AA_API_BASE`）。E2Eテストは[`e2e/`](../e2e/README.md)を参照。
